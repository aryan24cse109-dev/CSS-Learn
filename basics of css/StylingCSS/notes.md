----------------------------Styling--------------------------------------------------------
 
 Types of Styling :
 1) Inline Styling :
 A type of styling , in which you give style to recomended element in same html line , where you define the element .
 Syntax:
 For syntax , visit internalstyling file.

 2) Internal Styling :
 In this , we define styling and content in same file , for project and learning m this method is prefered , but not for professional .

 3)External Styling :
 In this , you create two differnet file for html and css :
 a) index.html for structure if content 
 b) index.css for styling .

 External styling is professional method of styling , you can also style like inetnak during production but after production you have to move styling content in different file of index.css .
