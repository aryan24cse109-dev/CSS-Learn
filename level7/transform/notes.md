<---------------Transform Notes----------->
Allows modification of an element's shape and position.
can perform operations like rotate , scale and translate .
Does not affect the layout of sorrounding elements.
Used to create visual effects like 3D space 
transformations.
 Implemented with functions like rotate(), 
scale(), and translate().


1) Transform : Rotate 
• Rotates an element around a 
fixed point.
• Defined using the rotate() 
function within the 
transform property. 
• Default rotation point is the 
element's center
• Default rotation axis is the z-axis.
To Roatate an element in specific direction, we can use the rotateX(), rotateY(), and rotateZ() functions.

2) Transform : Scale
• Scales an element's size along the x and y axes.
• Defined using the scale() function within the transform property.
We can use scaleX() and scaleY() functions to scale an element along a specific axis
In Scale we can use negative values to flip an element horizontally or vertically.
Scale means cahnging the size of an element without affecting its shape.


3) Transform : Translate
Is used to move an element from its original position along the x and y axes.
Defined using the translate() function within the transform property.
for specific axis we can use translateX() and translateY() functions.

4) Transform : Skew
• Distorts an element by tilting it along the x and y axes.
• Defined using the skew() function within the transform property.
For specific axis we can use skewX() and skewY() functions.
